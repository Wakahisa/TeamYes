IF roslyn 'access denied' errors
THEN IN (Menu)Project>Properties>Build Events>'Prebuild Event Command Line' text box
copy:

tasklist /FI "IMAGENAME eq VBCSCompiler.exe" 2>NUL | find /I /N "VBCSCompiler.exe">NUL
if "%ERRORLEVEL%"=="0" (taskkill /IM VBCSCompiler.exe /F) else (verify >NUL)

and paste into the text box.

This will kill the the OS process that did not correctly terminate. It will run every time before building/debugging.
Note that this may not save in the program options even if it says it did. That's why this file exists.
